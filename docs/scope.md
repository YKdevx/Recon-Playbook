# 📌 Project Scope

This repository focuses exclusively on reconnaissance.

## Included:
- Passive Recon
- Asset Discovery
- Prioritization
- Attack Surface Mapping
- Threat Modeling
- Active Validation (non-exploitative)

## Excluded:
- Exploitation
- Payload development
- Reporting workflows
- Vulnerability chaining

## Philosophy:
> This project stops at understanding the attack surface, not exploiting it.
